# Anexa A — Demonstrație video planIT Meals

## Cerință

Raportul trebuie să includă un **link funcțional** către o înregistrare video (3–5 minute) care demonstrează aplicația rulează end-to-end.

## Unde se inserează linkul

În `docs/RAPORT_SEMESTRUL_3.md`, secțiunea **6.3 Demonstrație video**, înlocuiți:

```
**Link:** *[inserați URL YouTube / Drive / Moodle]*
```

## Conținut minim al video-ului

| Minut | Scene | Ce trebuie vizibil |
|-------|-------|-------------------|
| 0:00–0:30 | Intro | Titlu proiect, URL aplicație (localhost sau deploy) |
| 0:30–1:00 | Autentificare | Register sau login reușit |
| 1:00–1:30 | Profil | Setare obiectiv caloric + restricție dietă (ex. vegetarian) |
| 1:30–2:30 | Plan alimentar | Generare plan 7 zile; scroll prin zile/mese |
| 2:30–3:15 | Shopping list | Deschidere listă cumpărături pentru plan |
| 3:15–3:45 | Frigider | Adăugare 2–3 produse; opțional sugestii rețete |
| 3:45–4:15 | Tracking | Înregistrare masă; total calorii zilnic |
| 4:15–4:30 | Rețete | Căutare/filtru; detaliu rețetă |
| 4:30–5:00 | Outro | Rezumat funcții; mențiune limbă RO/EN dacă e cazul |

## Checklist înainte de upload

- [ ] Backend și frontend pornite; date seed prezente (rețete vizibile)
- [ ] Rezoluție minimă 1280×720; audio opțional (narare sau subtitrări)
- [ ] Fără parole/token-uri vizibile în ecran
- [ ] Link accesibil pentru evaluator (ne-privat sau link Moodle)
- [ ] Link adăugat în raportul PDF final

## Instrumente recomandate

- macOS: QuickTime Player → New Screen Recording
- OBS Studio (cross-platform)
- Loom / ShareX (Windows)

## Exemplu formulare în raport

> Demonstrația video a aplicației planIT Meals este disponibilă la:  
> **https://youtu.be/XXXXXXXXXXX**  
> Durată: 4 min 12 s. Înregistrarea acoperă fluxurile de autentificare, generare plan, listă cumpărături și tracking.
